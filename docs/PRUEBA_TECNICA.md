# Prueba Técnica — Baile con la Muerte

Como parte del proceso de evaluación, deberás desarrollar una aplicación web para agendar una cita para “bailar con la muerte”. La aplicación debe permitir que un usuario seleccione una fecha y una hora disponible para reservar su cita.

## Reglas principales

- El usuario debe poder seleccionar una fecha. 
- Solo se pueden seleccionar días hábiles.
- No se pueden seleccionar fechas pasadas.
- El horario disponible es de 09:00 a 19:00.
- Las citas son por bloques de una hora. 
- Una misma fecha y hora no puede reservarse dos veces. 
- El usuario debe ingresar su nombre y correo electrónico. 
- Un mismo correo electrónico solo puede tener una cita registrada. 
- La aplicación debe mostrar qué horarios están disponibles y cuáles ya están ocupados.

## Tecnologías requeridas

- Backend: Laravel
- Frontend: Vue / Nuxt
- Base de datos: PostgreSQL

El proyecto debe incluir una breve documentación con los pasos necesarios para instalarlo, configurarlo y ejecutarlo correctamente. El objetivo no es crear una aplicación compleja, sino una solución funcional, clara y bien organizada.

Muchas gracias y éxito con la prueba.
